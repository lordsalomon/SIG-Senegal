# SIG-Senegal
For sharing GIS and Geospatial data about Senegal...

## FORMAT
### 1. Vector
For Vector layers, Geopackage (.gpkg) and comma-separated values (.csv) formats are preffered.
### 2. Raster
Geotif files are preffered.
